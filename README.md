# Online Banquet Booking System (OBBS)

The Online Banquet Booking System (OBBS) is a web-based application developed as a final year BCA project. The system allows users to book banquet services online while providing an admin panel to manage bookings, services, event types, and customer enquiries.

This project aims to simplify banquet booking management by providing an online platform where users can view services, register, and book events easily.

## Features

### User Features
- User registration and login
- View banquet services
- Book banquet services for events
- View booking history
- Update profile
- Change password
- Send enquiries through contact form

### Admin Features
- Admin dashboard
- Manage banquet services (Add / Delete)
- Manage event types
- Manage bookings (Approve / Cancel)
- Manage website pages (About Us, Contact Us)
- View customer enquiries
- Generate booking reports
- Search booking details

## System Modules

The system consists of two main modules:

1. Admin Module
2. User Module

Admin manages the entire system including services, bookings, events, and reports. Users can register, browse services, and book banquet events online.

## Database

The system uses MySQL database with multiple tables including:
- tbladmin
- tblbooking
- tbleventtype
- tblservice
- tbluser
- tblcontact
- tblpage
- states
- cities

## Purpose of the Project

The main objective of this project is to develop a centralized system that manages banquet booking events efficiently. It helps reduce manual work and allows users to easily book banquet services through an online platform.
